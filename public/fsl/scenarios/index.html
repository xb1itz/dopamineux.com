    FSL scenarios

    Scenarios describe the stories and context behind why a specific user or user group comes to your site.
    They note the goals and questions to be achieved and define the steps of how the users can achieve them on the site.

    Scenarios are critical both for designing an interface and for usability testing.
    They do not describe the details and components of site UI. However the prototype that follows will be highly based on them.

    Scenario index

    Here all possible scenarios are grouped by particular moments.

    Each scenario is tagged with confidence that reflect how likely particular scenario will be valuable for users for reaching their goals. Confidence scale is 1-5 where means the scenario is not likely to be valuable and means high return of investment.


    Onboarding scenarios

    These are scenarios designed to hook a new FSL user by providing them perceptive value as soon as possible.
    The challenge here is to present FSL as new platform that will be used by every football fan.

    • Organic visitor to landing page User who enters FSL landing page directly w/o any context
    • Download funnel from Store Scenario when user downoads app from store
    • Virality trigger Scenario designed to get users to infite friends to the app
    • Invite from friend User who receives a general invitation link from friend via any channel
    • Link from social media Users who see shared platform content on social media and clicks through
    • Invite to watch live score User who receives an invitation to check live game feed
    • SEO based acquisition Userts who find public content on search engines and clicks through
    • GAME based acquisition The acquisition based on gamification factors
    • Organic visitor with VP User who enters FSL platform directly and is provided with additional value before signing up.

    PRE Match

    Scenarios targeting users before particular match starts.

    • Upcoming matches Checking all upcomig matches
    • Match details Looking for game details and information on it (time, venue, stats)
    • Tickets Looking tickets to particular game
    • Predictor Posting a score prediction
    • Friend's predictions Checking predictions of friends and followed people for the same game
    • Other predictions Checking predictions of other fans for the same game
    • Predictions and tactics Checking free content of other users on particular game

    Advanced PRE Match

    Scenarios targeting users before particular match starts with additional gamification.
    Note that all these scenarios are subject for experiments due to need for balancing of gamification factors.

    • Create a challenge Creating a challenge based on prevoiusly posted prediction
    • Share challenge Sharing created challenge publically via available share channels
    • Inviting friends challenge Inviting FSL friends to acceps created challenge
    • Starting XI Posting starting layout or tactics

    Match time scenarios

    Scenarios happenig during a match.

    • Live score Checking a live score of the match
    • Live broadcast (if possible) Watching a live brodcast of the game
    • Match events Checking a special match events (goals, cards, subs)
    • Live reactions Checking live reaction of other FSL users
    • React Post a rection/comment in the context of the match
    • Invite friends Inviting friends to watch a live score
    • Share a moment Sharing a special moment of the game (goals, cards, subs) via available share channels
    • Share a current result Sharing current score of the game via available share channels

    Match time scenarios IRL

    Scenarios happenig during a match, watching it in real life.

    • Selfie Adding a photo/selfie with geotag
    • Invite Inviting other fans to watch match IRL
    • Event/radar Creating an IRL watch event publicly accessible to everyone

    POST Match

    Scenarios targeting users after particular match ends.

    • Check score Checking a score of ended match
    • Check reactions Checking reactions of other fans after match ended
    • Key moments Checking a key moments of the match
    • Reaction Post a public post about the match
    • Rate Post a detailed rating for team players for particular match
    • Share score Sharing match score via available share channels
    • Sharing reaction Sharing a positive or negative reaction about match publicly

    Advanced POST Match

    Scenarios targeting users after particular match ended with additional gamification.
    Note that all these scenarios are subject for experiments due to need for balancing of gamification factors.

    • Prediction reward Receiving elevation for correct predictions
    • Starting XI reward Receiving elevation for correct for layout/tactics
