Scenarios:

    Match time scenarios IRL

    Scenarios happenig during a match, watching it in real life.

    Legend

    Each scenario represents a particular business case. What steps users take to interact with the system to complete their goals.
    Some of the steps have additional indicators that are explained below:

    • A trigger

      Represents a point or event where users are being engaged into the scenario.

    • Investment

      A step where a user has to spend mental effort to proceed to further steps of the scenario. This is usually the most challenging moment in the scenario.

    • AFK timeout

      Away From Keyword moment that represents a temporary break in the scenario, where users are not actively engaged in a system.

    • Notification

      Represents an automatic notification that engages user outside of a system.

    • The reward

      A motivator that rewards user for completing whole (or a part of) scenario.

    #1 Selfie

    • 1. Entry point

      The scenario begins a particular match starts.

    • 2. Finding a match

      Match can be found in different ways:

      • Search feature in navigation
      • User's team feed
      • Match list

    • 3. Take a photo CTA

      System displays CTA to take a photo and upload to match

    • 4. Posting a photo

      Scenario cointinues using the same mechanics as sharing simple photo in Creation scenarios

    #2 Invite

    • 1. Entry point

      The scenario begins a particular match starts.

    • 2. Invite friends to watch IRL

      User can post an invitation to watch match IRL using same share mechanics, as sharing a reaction or match itself.

    #3 Event/radar

    • 1. Entry point

      The scenario begins a particular match starts.

    • 2. Finding a match

      User navigates to match same way as in Scenario #1

    • 3. Create public invitation CTA

      CTA that opens special invitation form where user can enter details of invitation

    • 4. Submit infitation

      User fills out and submits invitation

    • 5. Success message

      Success message is displayed.
      An invitation post appears on users feed.
      Invitation appears in match reaction feed publicly.

      5.1. Initation notification

      User's followers are notified about invitation and can join user to watch match IRL.
